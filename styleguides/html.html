Standard HTML Declaration

Aturan dalam deklarasi HTML

  • Gunakan kutip 2, bukan kutip 1.
  • Urutan penempatan attribute:
    1. Class
    2. ID
    3. Attribute bawaan (href, src)
    4. Attribute global
    5. Data attribute
  • Apabila deklarasi tag sudah terlalu panjang, diperbolehkan untuk menambahkan new line untuk tiap attribute.
<!-- Bad -->
<a data-attribute="link" href="..." id='link' class="link" title="my link">Link</a>

<!-- Good -->
<a class="link" id="link" href="..." title="my link" data-attribute="link">Link</a>

<a
    class="link"
    id="link"
    href="..."
    title="my link"
    data-attribute="link">Link</a>

Penamaan class dan ID

Penamaan class dan ID hanya diperbolehkan menggunakan:

  • Alphabet a-z || A-Z
  • Numeric 0 - 9
  • Dash '-'
  • Underscore '_'
  • Harus berawalkan alphabet

Aturan penamaan untuk:

  • Class: tidak boleh camelcase dan harus lowercase (kecuali bawaan plugin)
  • ID: harus camelcase
<!-- Bad -->
<a class="myClass" id="my-class" href="..."></a>

<!-- Good -->
<a class="my-class" id="myClass" href="..."></a>
<a class="my-very-long-class" id="myVeryLongClass" href="..."></a>

Class naming convention

Penamaan class haruslah se-abstract mungkin agar style yang sudah dibuat dapat digunakan berkali-kali.

Intinya buatlah nama class dengan pertanyaan "untuk apa", bukan "untuk siapa".

Gunakan modifier untuk meng-achieve hasil visual yang diinginkan. Apabila memang terpaksa membuat class untuk 1 element tertentu, gunakan modifier.

<!-- Bad -->
<!-- Contact has smaller width -->
<div class="container-contact">...</div>

<!-- Download button has large padding -->
<button class="btn btn-download">download</button>

<!-- Good -->
<div class="container container--small">...</div>
<button class="btn btn--large">download</button>

Penggunaan tag <a> dan <button>

Sederhananya, gunakan tag <a> apabila:

  • Ingin berpindah ke halaman lain
  • Menuju ke section tertentu pada sebuah Web. Contoh: <a href="#top">Go to top</a>

Lalu gunakan <button> apabila pada saat di-click, sebuah action akan berjalan tanpa pindah halaman. Contoh penggunaan <button>:

  • Trigger menu
  • Button accordion

Label untuk tiap input

Secara semantic dan accessibility, tiap input harus memiliki label, meskipun secara design label digantikan oleh placeholder.

Contoh: Form tanpa label

<div class="form-row">
    <!-- class sr-only untuk menyembunyikan label, namun tetap accessible melalui screen reader -->
    <label class="sr-only" for="exForm1">Username</label>
    <input class="form-input" id="exForm1" type="text" placeholder="Username">
</div>
<div class="form-row">
    <label class="sr-only" for="exForm2">Password</label>
    <input class="form-input" id="exForm2" type="password" placeholder="Password">
</div>
