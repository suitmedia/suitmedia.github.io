Slick Carousel

Digunakan apabila membutuhkan image slider ataupun carousel.

Documentation
kenwheeler.github.io/slick/
Version
1.3.15
CLI
bower i slick-carousel#1.3.15 --save
Bower registry
"slick-carousel": "~1.3.15"
Note

Secara default, slick carousel sudah memberikan style untuk arrow dan pagination/indicator. Apabila ingin mengubah style arrow dan indicator, disarankan jangan meng-override style yang sudah ditetapkan oleh Slick dikarenakan deep nesting yang dibuat oleh Slick.

Apabila dipaksakan meng-override style, akan jauh lebih sulit dan harus menggunakan keyword !important yang di mana tidak dianjurkan.

Yang perlu dilakukan adalah:

  1. Buka file slick.css
  2. Hapus line 26-57

Setelah itu, baru declare style yang diinginkan di file _plugin.scss.
